# AI Assistant Guidelines

This is a Laravel 13 and Vue.js (Inertia) project.

## Specialist Routing

- Use `@laravel-specialist` for all backend tasks, including routing, modular controllers, services, database queries, and MCP MySQL interactions.
- Use `@vue-specialist` for all frontend tasks, including Vue components, PrimeVue integration, and Tailwind CSS adjustments.

## Core Rules

1. Always follow the project's `.gitignore` rules.
2. Strictly adhere to the Layered Architecture guidelines defined in the specialist prompts.
3. Review existing module structures in `app/` and `resources/js/` before generating new files.
4. Keep the codebase self-documenting; avoid adding unnecessary inline comments in the generated code.
